# webpack-using-js-html-sass

1. Clone project
2. Install project dependencies
   * Install node

     ```
     $ sudo apt-get update
     $ sudo apt-get install nodejs
     ```
   * Install node packages

     ```
     $ npm install
     ```
3. Done with installation.

# How to use
   * To run local server

     ```
     npm start
     ```
   * To create build

     ```
     npm run build
     ```
